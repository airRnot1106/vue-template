## How to use

```bash
$ git clone git@github.com:airRnot1106/vue-template.git
```

```bash
$ cd vue-template
```

```bash
$ yarn install
```

## Development

```bash
$ yarn run dev
```

## Build

```bash
$ yarn run build
```
